QUICK REFERENCE
===============

Installation:
  pip install -r requirements.txt

Test Setup:
  python scraper/test_scraper.py

Scrape All Data:
  python scraper/scrape_10k.py

Scrape Completed Only:
  python scraper/scrape_completed.py

Scrape Incomplete Only:
  python scraper/scrape_incomplete.py

Analyze Data:
  python scraper/analyze_data.py


OUTPUT FORMAT
=============

Each JSON file contains:
  - metadata (file info, timestamp, source)
  - projects array (up to 1000 project records)

Each project record includes:
  - id, projectName, projectCode, description
  - latitude, longitude, streetAddress
  - city, barangay, province, region
  - status, cost, utilizedAmount
  - implementingOffices, contractors, sourceOfFunds
  - dateStarted, completionDates
  - and much more...
